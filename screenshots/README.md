# Ekran Görüntüleri

Bu klasöre uygulamanın ekran görüntülerini ekleyin.

## Gerekli Ekran Görüntüleri:

1. `home_page.png` - Ana sayfa (ürün listesi)
2. `detail_page.png` - Ürün detay sayfası
3. `cart_notification.png` - Sepet bildirimi

## Ekran Görüntüsü Alma:

### Android Emulator:
- Emulator yan panelindeki kamera ikonuna tıklayın
- Veya klavyeden Ctrl+S (Windows/Linux) veya Cmd+S (Mac)

### iOS Simulator:
- Cmd+S tuşlarına basın
- Veya File > Save Screen

### Fiziksel Cihaz:
- Android: Power + Volume Down
- iOS: Power + Volume Up

Ekran görüntülerini aldıktan sonra bu klasöre kopyalayın.
